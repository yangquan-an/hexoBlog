Git 学习

2024/7/9 总结 git 技术栈

Git的概念

分布式的版本管理工具,提高多人开发的效率

本地生成 Git 仓库的方式

  1. git init
  2. git clone https://…(远程仓库网址)
  3. git status 检查当前目录是否是git仓库
  4. ls -a 可以查看到隐藏的 .git 文件夹

Git 基本流程

git status 分类

查看提交的流程

  1. git log 查看详细的流程信息
  2. git ls-files 查看暂存区的内容
  3. git log --oneline 查看简洁的流程信息
  4. git reflog 查看历史记录，可以回溯操作

git reset 的三种模式

git reset –soft(hard | mixed) 回退的版本 id;操作失误后可以通过 git reflog查看历史记录进行回溯

git diff

git diff 可以对工作区跟暂存区的内容进行比较
git diff 版本id 版本id2 可以对两个版本进行比较

远程仓库

本地仓库远程连接远程仓库

  1. git remote add origin git@github.com:yangquan-an/gits.git(远程连接)
  2. git remote -v 查看远程连接的仓库
  3. 由于新建的远程仓库是空的，所以要加上-u这个参数:git push -u origin master:master(本地:远程)
